PROJECTS

  • LIVE ART

    Touch image for more info!

    A response to the DEV community 'Grant for the Web' hackathon centered around web monetization. Live Art is a full stack web app that allows supporters to view a live art canvas and make micro payments direct to the artist. Both the front-end and back-end of the project are hosted with Heroku.

    Built remotely, alongside another developer, VS Code Live Share and Git Hub Projects were used for effective collaboration and agile development.

    Tech Stack

    JavaScript | React | Socket.io | Canvas API | Express | Node | MongoDB | Mongoose

  • RECHARGE

    Touch image for more info!

    A full stack mobile app built to assist owners of electric cars in their journey planning. Users can generate a route planner based on how often they need access to a charge point. The app also allows the user to view ammenities local to each charge point.

    reCharge was a collaborative project, created with three other Northcoders students during the final two weeks of the bootcamp and using agile practices.

    Tech Stack

    Dart | Flutter | Google Maps API | Rapid Charge API | Ruby | Rails | PostgresSQL

  • TOPICAL

    Touch image for more info!

    A full stack Reddit style news web app made during the Northcoders 'Developer Pathway' bootcamp.

    The back-end of the project is a RESTful API built using extensive TDD(Test Driven Development) and a relational database. The front-end is a single page application using components to create the effect of rendering multiple 'pages'. The project is hosted with Heroku on the back-end and Netlify on the front-end.

    Tech Stack

    JavaScript | React | Mocha | Chai | Express | Node | PostgresSQL | Knex
